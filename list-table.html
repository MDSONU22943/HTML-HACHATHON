Welcome to HTML Lists & Tables Tutorial

This tutorial explains in the simplest way how to create Lists, Nested Lists, and Tables in HTML.

1. Simple List

A simple list is used to show related items in bullet points. Each item is independent and equal.

  • Apples
  • Bananas
  • Oranges

Explanation:

  • <ul>: Stands for Unordered List (bullets by default).
  • <li>: Each item in the list.

2. Nested List (List Inside a List)

A nested list helps organize information in a hierarchy, making it easier to show categories and sub-categories.

  • Fruits
    • Apples
    • Bananas
    • Oranges
  • Vegetables
    • Carrots
    • Spinach
    • Broccoli
  • Dairy Products
    • Milk
    • Cheese
    • Yogurt

Why Use Nested Lists?

  • Organizes content better by showing sub-items related to a main item.
  • Useful in displaying categories, topics, and step-by-step guides.

3. Simple Table

A table helps present structured data clearly using rows and columns, like a spreadsheet or timetable.

Day Morning Task Evening Task
Monday Go for a run Read a book
Tuesday Attend work meeting Watch a movie
Wednesday Write code Cook dinner

Table Structure Explained:

  • <table>: Main container for the table.
  • <thead>: Table header section (column titles).
  • <tbody>: Holds the table’s data rows.
  • <tr>: Table row (horizontal section).
  • <th>: Table header cell (bold, centered text).
  • <td>: Table data cell (regular content).
