Easy HTML Guide for Beginners

This guide will help you understand what HTML is and how to use basic HTML tags in a very simple way.

What is HTML?

HTML stands for HyperText Markup Language.

  • HyperText: Links that connect web pages together (like clicking links on google.com).
  • Markup Language: Tags that give structure to your web content (like paragraphs, headings).

Headings

Headings define the structure of your page from most important (<h1>) to least important (<h6>).

This is Heading 1 (Main Title)

This is Heading 2 (Major Section)

This is Heading 3 (Sub Section)

This is Heading 4

This is Heading 5
This is Heading 6

Note: Headings are not just to make text big, but to give structure and help search engines understand your content.

Paragraphs

Use the <p> tag to group sentences together into a paragraph.

This is the first paragraph. It contains sentences about a topic.

This is a second paragraph. Each paragraph is automatically spaced by the browser.

Horizontal Line

The <hr> tag adds a horizontal line to separate content sections.

This is one topic.

This is a completely different topic separated by a line.

Line Break

The <br> tag forces text to go to the next line without starting a new paragraph.

Example:
221B Baker Street
London, England

Lists in HTML

What is a List?

A list is a way to group related items together in an organized format. There are two main types of lists in HTML:

Unordered List (Bullets)

An <ul> tag creates a list where each item has a bullet (●). Use it when the order of items doesn’t matter, like shopping items.

Example: Things needed to make tea

  • Tea Bag
  • Water
  • Sugar
  • Milk

Explanation: This list shows simple items needed. The browser automatically puts a bullet before each item.

Ordered List (Numbers)

An <ol> tag creates a numbered list (1., 2., 3., ...). Use it when the order matters, like steps in a recipe or instructions.

Example: How to make tea step by step

  1. Boil water in a kettle.
  2. Add tea bag, sugar, and milk into the water.
  3. Boil for 5 minutes.
  4. Pour the tea into a cup and enjoy.

Explanation: The browser automatically numbers each step to show the correct order of actions.

Why Use Lists?

  • They help organize information clearly.
  • Easy for readers to understand grouped data.
  • Important for creating clean, structured webpages.

Links (Anchor Tag)

Use <a> to create clickable links.

Example: Visit Coder Army for more tutorials.

Tip: Use target="_blank" to open the link in a new tab.

Images

Use <img> to display an image.

Attributes:

  • src: Path to image URL.
  • alt: Description for screen readers or when image fails to load.
