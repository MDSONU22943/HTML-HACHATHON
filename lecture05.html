Simple HTML Media Guide for Beginners

Learn how to embed videos and audio into your webpage the easy way, without relying on plugins like Flash.

Why Add Media?

The web is more than just text and pictures. We need video and audio to make websites interactive (like playing a tutorial video or listening to music).

The Basic Video Example

Problem: A video tag without controls only shows the first frame as a static image.

Solution: Add the controls attribute to show a play/pause button, timeline, and volume control.

Control Video Behavior with Attributes

  • autoplay: Starts playing the video automatically when the page loads (often muted).
  • loop: Plays the video over and over again.
  • poster: Displays an image before the video starts (like YouTube thumbnails).

Best Practice – Multiple Formats for Maximum Compatibility

Why? Different browsers support different video formats.

Solution: Use multiple <source> tags inside the <video> tag.

How It Works:

  • The browser tries the first source (webm), if supported, it plays.
  • If not, it moves to the next source (mp4).
  • If neither works, fallback text appears.

Accessibility – Adding Captions and Subtitles

Problem: Deaf users or non-native speakers can't understand the video.

Solution: Add <track> tags to provide captions or subtitles.

Comparing Two Ways to Embed Video

Feature Multiple Sources (Best) Single Source (Simple)
Browser Compatibility Excellent – Provides fallback formats Good – May not work everywhere
Flexibility High – Can list many file formats Low – Only one format possible
Simplicity More complex (multiple lines) Very simple (one line)
Fallback Text Yes – Custom message if unsupported No fallback
